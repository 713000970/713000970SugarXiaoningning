📋 通用规则

🧪 预处理
1、预上传，测试是否重复。若有问题：
（1）资料缺失（兼职——晶晶——运营，在一周内给确切处理结果，若不处理，则任务挂起）
（2）资料重复（运营核实版权归属——晶晶——下线/退稿）涉及重复率标准，出版物/非出版
✂️ 拆分
1、PDF按目录，若无，按标题，杜绝单页
2、PPT检查删除敏感信息（联系方式、二维码等），然后直接上传
3、WORD检查删除敏感信息（联系方式、二维码等），然后直接上传
⚠️ 备注：以上三种格式有存在主书+小册子的情况，小册子拆分后，需单独建立专辑上传
🏷️ 命名
1、专辑命名
（1）【品牌】学年 + 年级 + 学科 + 对应封面名称（版本）
（2）【品牌】学年 + 年级 + 学科 + 对应文件名称（版本）
2、资料命名
（1）具体标题（第X章+第X节+标题名）-专辑名
（2）若资料名与书籍名重复，则无需后缀
📁 专辑规则
1、专辑目录
（1）有目录书籍 → 1:1 还原，资料按顺序排序
（2）同步书籍 → 获取主数据目录，资料放对应目录下
2、专辑简介
XXXX（书籍名称）系列资料由XXXX（提供者名称）提供，并授权学科网在互联网进行发布，侵权必究！
3、专辑封面
（1）无封面 → 使用去年同期产品封面
（2）去年无 → 智能平台生成，要求最下标必须写 学科网书城，点击制作封面 →
4、其他
（1）专辑"热点""推荐"等该栏标签 不做任何勾选
💰 上传规则
1、定价
（1）参考定价文档：
📄 点击查看定价文档 →
2、发布时间
（1）专辑：立即发布
（2）资料：立即发布1/10，剩余按教学进度发布，一般以月为颗粒度同步

⚠️ 特例 · 试卷 / 中职 / 音视频
真题试卷非教辅版权试卷
• 高考：不处理上传
• 中考 / 小升初：发布后在 XMP 后台加入网校通
普通真题卷校考 / 省市区县统考卷
资料打标作者 "教辅精选"，发布后在 XMP 后台加入网校通
中职产品 发布后在 XMP 后台加入网校通
音视频 勾选 禁止下载 标签
